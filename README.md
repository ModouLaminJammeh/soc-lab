# 🛡️ SOC Lab

## 📌 Objective
My goal with this SOC Lab is to build a realistic environment where I can practice detecting and analyzing cyber attacks. Instead of relying only on theory, I focus on setting up systems, generating logs, and understanding how attacks actually appear from a defender’s point of view. This lab helps me gain hands-on experience with monitoring, log analysis, and basic incident response.

---

## 🧠 Skills Learned

- Practical experience working with SIEM tools for monitoring and analysis  
- Ability to read and interpret logs from multiple sources  
- Understanding how common attack patterns appear in real environments  
- Improved knowledge of network behavior and potential vulnerabilities  
- Stronger analytical and problem-solving skills through investigation  

---

## 🧰 Tools Used

- SIEM platform for log collection and analysis  
- Virtual machines (lab environment)  
- Network monitoring and packet analysis tools  
- Attack simulation tools for generating security events  

---

## ⚙️ Lab Setup Overview

This lab simulates a small SOC environment where:
- Logs are generated from different systems  
- Data is ingested into a SIEM  
- Attacks are simulated to produce real telemetry  
- Events are monitored and analyzed for suspicious activity
